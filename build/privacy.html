Privacy Policy

Last updated: July 10, 2026

Welcome to NeverSEO. We are committed to protecting your privacy and personal data. This policy explains how we collect, use, and protect information when you use our web application.

1. Information We Collect

When you create an account or use NeverSEO, we may collect:

  • Identification information: Name, email address, avatar (when logging in via Google OAuth).
  • Usage data: Activity logs, AI Token usage.
  • Data provided by you: SEO campaigns, keyword lists, content drafts, third-party AI service API keys (securely encrypted using AES-256 in our database).

2. Compliance with Google API Data Policy

NeverSEO's use and transfer to any other app of information received from Google APIs will adhere to Google API Services User Data Policy, including the Limited Use requirements.

Specifically, when you grant access to Google Search Console or Google Analytics for Audit and measurement features:

  • We only use this data to provide or improve features for users (ranking analysis, traffic).
  • We do not share or sell user data to third parties for advertising or any other unrelated purposes.
  • We do not allow humans to read this data unless we have your explicit consent for technical support or for security/legal compliance purposes.

3. Data Storage and Security

The system supports multiple accounts, and each working group (Organization) has an independent data space. Sensitive data (such as API Keys, Access Tokens) is securely stored using the most advanced encryption algorithm (AES-256).

4. Your Rights

You have the right to access, update, or request the deletion of all your personal data from the system. Deleting an account (Organization) will permanently delete all associated project data, configurations, and content.

5. Contact

If you have any questions about this Privacy Policy, please contact us via email at support@neverseo.com.
